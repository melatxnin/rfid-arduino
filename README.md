# Arduino RFID Acces System (WOKWI)

Simulation of RFID access control system using Arduino.

## Components

- Arduino Uno
- MFRC522 RFID Reader
- Resistor 220Ω (2)
- LED (2)
- Buzzer

## Features

- Reads RFID card UID
- Compares the UID with the authorised one
- Green LED and high sound when access is granted
- Red LED and low sound when access is denied
- UID displayed in the Serial Monitor

## Controls

- Click on the RFID reader
- Choose one of the available cards at the top of the simulator
